| Acronym | Meaning | Notes |
|---------|---------|-------|
| LOS     | Line Of Sight | | 
| ATCF    | As The Crow Flies | | 
| AGL     | Above Ground Level | | 
| HWT     | HeyWhatsThat | A website that creates Line Of Sight signal coverage maps |
| SNR     | Signal-to-Noise Ratio | How much unrelated radio interference there is compared to meshtastic signals | 
| RSSI    | Received Signal Strength Indicator | An abstract measurement of how strong a received signal was | 
| SWR     | Standing Wave Ratio | How well tuned an antenna is, closer to 1 is better | 
| BBS     | Bulletin Board System | A local message board hosted on a Meshtastic node that users on the mesh can post to and read from | 
| MQTT    | Message Queueing Telemetry Transport | A general purpose internet-based data streaming format that Meshtastic uses to bridge from the mesh to the internet | 
| Rx      | Receive | A radio's ability to hear radio waves |
| Tx      | Transmit | A radio's ability to create radio waves |
| ChUtil  | Channel Utilization | The percentage of time the radio channel is busy |
| LoRa    | Long Range - A low-power | The radio standard that Meshtastic uses | 
| GPIO    | General-Purpose Input/Output - Configurable pins on microcontrollers | | 
| 4631    | RAK's 4631 "WisBlock" node | | 
| RAK150  | A completed node built around RAK's 150x75mm solar enclosure | |
